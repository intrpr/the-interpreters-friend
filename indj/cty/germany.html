Explanation of glossing system used
(to understand how I describe the signs in this dictionary)

Aachen (previously named Aix-la-Chapelle)
(2h)[A dot], POs ><, FOs away, thumbs are tilted > DS and hands descend in a slant > NDS. For a static gif of this sign, see Surdimobil - Dictionnaire des signes. Click on "Aix-la-Chapelle". It will look like nothing has happened, but scroll down to see the illustration.

Augsburg (Augsburg)
Loose [1] taps under the eye several times.1 (< AUGE which means "eye".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Augsburg.

Berlin (Berlin)
[S], PO > signer is held at temple, twists away, and backhand returns to touch temple.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Berlin. For a QuickTime movie of this sign, see Woordenboek Nederlands - Vlaamse Gebarentaal: Berlin.

Bochum (Bochum)
(2h)[S], POs towards, NDH is held at waist and HD strikes it from above.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Bochum.

Bonn (Bonn)
[V], PO towards, FO > NDS is held at corner of eye.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Bonn.

Braunschweig (Braunschweig)
The palm of [A] is rubbed in a circle against the corner of the mouth.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Braunschweig.

Bremen (Bremen)
[L:], PO towards is tapped twice against chin.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Bremen.

Bremerhaven (Bremerhaven)
[L:], PO towards, is tapped twice against chin, then [5], PO down, moves down and closes to an [S].1 (< BREMEN plus HAFEN, which means "port".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Bremerhaven.

Buxtehude (Buxtehude)
[N], PO away, moves in a CCW circle.1 (< .) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Buxtehude.

Celle (Celle)
Thumb of [A dot] is drawn up the DS of the chest.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Celle.

Cologne (Köln)
(1) [loose C] is held at temple and then twists away into an [S].1 (2) (2h)[C] POs ><, are held at side and then ascend while closing into (2h)[S].1 (< cathedral spires.) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Köln.

Dortmund (Dortmund)
[1] points to mouth and circles CCW.1 (< MUND, which means "mouth".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Dortmund.

Dresden (Dresden)
[5:] strikes chest once.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Dresden.

Duisburg (Duisburg)
[1] twists twice against cheek.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Duisburg.

Dusseldorf (Düsseldorf)
(1) [modified X], PO > NDS, where the thumb is held against the bent forefinger, is held high and brought forward in an arc.1 (2) [modified X], PO up, where the thumb is held against the bent forefinger, is held at side and circles CW.1 For a static gif of these signs, see Allgemeines Gebärdenlexikon: Düsseldorf.

Eckernfoerde (Eckernförde)
(2h)[B], PO towards, strike FTs.1 (< ECKE, which means "corner".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Eckernförde.

Erfurt (Erfurt)
[A dot], PO away, is held high at side and moves in an arc towards NDS.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Erfurt.

Essen (Essen)
[A] or [bX], PO towards, moves towards mouth twice.1 (< ESSEN which means "eat".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Essen.

Frankfurt (Frankfurt)
(2h)[bX], POs ><, FOs away, hands alt. move to and fro, like one of the variants of the ASL sign CONTROL.

Fuerth (Fürth)
[open C] with thumb at mouth and fingers at forehead moves forward while closing to a [O^].1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Fürth.

Goettingen-Heide (Göttingen-Heide)
(2h)[B] are held against each other as in prayer and move in an up and down circle CW.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Göttingen-Heide.

Halberstadt (Halberstadt)
(2h)[K], POs ><, are held with DH over NDH and move up and down together twice.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Halberstadt.

Halle (Halle)
[B^] wipes chin from NDS to DS.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Halle.

Hamburg (Hamburg)
[flat C] is held at temple and moves to DS while closing to [O^].1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Hamburg.

Hameln (Hameln)
(2h)[5:] are held at mouth and fingers are fluttered as if playing a flute.1 (< the legend of the Pied Piper of Hameln.) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Hameln.

Hannover (Hannover)
(1) [V], PO away, moves down in a curvy pattern.1 (2) [bC], PO > NDS, waves across face twice.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Hannover.

Husum (Husum)
[Bb] is brushed against side of mouth twice.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Husum.

Itzehoe (Itzehoe)
[I],PO > NDS, FO up, makes small circles CCW.1 (< initialized sign.) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Itzehoe.

Kaltenkirchen (Kaltenkirchen)
(2h)[S] are held high at sides and moves as if signer is shivering plus (2h)[B] move upwards diagonally towards one another and touch FTs, indicating slanted roof.1 (< KALT and KIRCHE which means "cold church".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Kaltenkirchen.

Kiel (Kiel)
FT of [1] strikes Adam's apple.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Kiel.

Leipzig (Leipzig)
[bC] runs FTs along chest as if indicating a sash that goes from NDS shoulder to DS hip.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Leipzig.

Mainz (Mainz)
[B] runs pinkyside along chest as if indicating a sash that goes from NDS shoulder to DS hip.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Mainz.

Munich (München)
[5:] touches crown of head and moves upwards.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: München.

Munster (Münster)
[modified X] twists at cheek, where the thumb rests on the bent index finger.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Münster.

Neumuester (Neumüster)
[modified X] taps cheek, where the thumb rests on the bent index finger.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Neumüster.

Osnabrueck (Osnabrück)
[bC], PO away, moves in upward arc > DS.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Osnabrück.

Pinneberg (Pinneberg)
[F] taps top of [loose B], PO down, plus [5], PO down, moves forward in upward arc, indicating a mountain.1 (second part of sign < BERG which means "mountain".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Pinneberg.

Ratzeburg (Ratzeburg)
[5:] draws FTs down top of [B], PO down.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Ratzeburg.

Recklinghausen (Recklinghausen)
[A dot] touches DS of chest and twists down at wrist.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Recklinghausen.

Rendsburg (Rendsburg)
[bC] touches NDS of chest and then moves in arc and touches DS.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Rendsburg.

Rostock (Rostock)
(2h)[S], POs down, touch at sides and separate.1 (< STOCK which means "stick"?) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Rostock.

Stade (Stade)
(2h)[A dot] touch thumbs and separate while twisting downward.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Stade.

Stuttgart (Stuttgart)
Thumb and index finger grasp nose with PO > NDS, then move away while closing into [bO^].1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Stuttgart.

Wiesbaden (Wiesbaden)
[B], FT > NDS, stroke chest up and down.1 (< BADEN which means "bathe".) For a static gif of this sign, see Allgemeines Gebärdenlexikon: Wiesbaden.

Wuerzburg (Würzburg)
Index finger snaps against thumb while PO > away.1 For a static gif of this sign, see Allgemeines Gebärdenlexikon: Würzburg.

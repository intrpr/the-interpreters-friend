EXPLANATION OF THE GLOSSING SYSTEM
(to understand how I describe the signs in this translation).

||: LORD, PLEASE ALLOW MY HEART SEE.
PLEASE ALLOW MY HEART SEE.
ME WANT SEE YOU-ctr.
ME WANT SEE YOU-ctr. :||

ME WANT SEE YOU-ctr ADVANCED FAR-AWAY.
YOUR-ctr GLORY BRIGHT, SHINY.
(<) YOUR-ctr POWER, (>) YOUR-ctr L-O-V-E, POUR-OUT
WHILE WE SING HOLY++'lift a bit higher each time'.

LORD, PLEASE ALLOW MY HEART SEE.
PLEASE ALLOW MY HEART SEE.
ME WANT SEE YOU-ctr.
ME WANT SEE YOU-ctr.

HOLY++'lift a bit higher each time'.
HOLY++'lift a bit higher each time'.
HOLY++'lift a bit higher each time'.
ME WANT SEE YOU-ctr.

Notes: The signs between "||:" and ":||" are to be repeated.

ME - Underlined words are honorific indices ("indexes"), using the FT of the upturned palm to indicate who is being honored.

ADVANCED - (2h)[B^], POs ><, FOs up, move upwards arcing towards the signer. For an animated gif of this sign, see Animated dictionary of religious signs - Deaf Missions: Exalted.

FAR-AWAY - [1], twists while moving away with tongue shuttling from side to side. For a QuickTime file of this sign, see ASL Pro: Far.

GLORY - (2h)[5], POs><, FOs away from signer, DH which is on top wiggles fingers while ascending. For an animated gif of this sign, see Animated dictionary of religious signs - Deaf Missions: Glory.

SHINY - (2h)[open 8], POs down, FOs away, twist rapidly at wrists while rising. For a QuickTime file of this sign, see ASL Pro: Shine.

POUR-OUT - (2h)[C] hold an imaginary jar and pour it out.
